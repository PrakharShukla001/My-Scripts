## 📖 About This Repository

> **My-Scripts** ek collection hai useful automation aur configuration scripts ki — mainly **Docker**, **Kubernetes**, aur **Linux** ke liye. Yahan main time ke saath nayi scripts add karta rahunga jo real-world DevOps tasks ko simplify karti hain.

Ye repo un logon ke liye helpful hai jo:
- 🐳 **Docker** setup aur containers manage karna chahte hain
- ☸️ **Kubernetes** clusters configure karna chahte hain
- 🐧 **Ubuntu/Linux** systems ko automate karna chahte hain

## ⚙️ How to Use

```bash

git clone https://github.com/PrakharShukla001/My-Scripts.git

# Folder mein jao
cd My-Scripts

# Script ko executable banao
chmod +x script_name.sh

# Run karo
./script_name.sh
```
